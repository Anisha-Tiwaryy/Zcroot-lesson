A mini interactive lesson platform built as part of the zcroot.ai AI/ML Internship task.

Live demo: https://zcroot-lesson-r5yz.vercel.app/

What it does:
Two full lessons — Data Structures & Algorithms and Machine Learning Fundamentals — each with an AI avatar video powered by HeyGen, followed by an interactive quiz that unlocks after watching. The quiz gives instant feedback on each answer, tracks your score, and flows into the next lesson automatically.

Features:
HeyGen AI avatar videos embedded for both courses
Unlock & Start Quiz button activates after watching the video
Interactive quiz with multiple choice questions and instant feedback
Score card with performance summary at the end
Next lesson flow between DSA and ML
Video auto-stops when switching between lessons
Retry option to rewatch and redo the quiz

Courses:
Lesson 1 — Data Structures & Algorithms
Lesson 2 — Machine Learning Fundamentals

Tech used:
HTML, CSS, JavaScript
HeyGen AI Avatar Video Embedding
Deployed on Vercel
